# CapitalizeAllTextNotesApp

Revit Version: 2019.0
Type: ExternalCommand

Summary: 
Плагин делает весь текст в текстовых примечания (TextNotes) во всей модели ЗАГЛАВНЫМИ БУКВАМИ.

Описание: 
Эта команда делает следующее:
- Находит все Текстовые примечания (TextNote) в модели
- Меняет весь текст внутри TextNote на заглавные буквы

Инструкция: 
1.	Откройте модель или семейство, которое содержит TextNotes
2.	Запустите команду
